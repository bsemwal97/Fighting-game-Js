# 🥋 JavaScript Fighting Game

A browser-based 2D fighting game built using **HTML5, CSS3, and JavaScript**. This project demonstrates core game development concepts such as sprite animations, collision detection, health management, gravity, attack mechanics, and keyboard controls.

## ✨ Features

* 🥷 Two playable fighters
* ⚔️ Attack and collision detection
* ❤️ Health bar system
* ⏱️ Countdown timer
* 🏆 Win/Lose detection
* 🎞️ Smooth sprite animations
* 🌍 HTML5 Canvas rendering
* ⌨️ Keyboard controls
* 💥 Gravity and jumping mechanics
* 🎯 Hitbox-based combat

Simply open `index.html` in your browser.

## 🎮 Controls

### Player 1

| Action     | Key   |
| ---------- | ----- |
| Move Left  | A     |
| Move Right | D     |
| Jump       | W     |
| Attack     | Space |

### Player 2

| Action     | Key         |
| ---------- | ----------- |
| Move Left  | Left Arrow  |
| Move Right | Right Arrow |
| Jump       | Up Arrow    |
| Attack     | Down Arrow  |

> Update the controls if your implementation uses different keys.
