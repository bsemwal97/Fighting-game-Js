# 🥊 Fighting Game JS

A 2D fighting game built using HTML5 Canvas and JavaScript featuring real-time combat, character animations, health bars, collision detection, and a countdown timer.

## 🎮 Features

* Player vs Player combat
* Attack and hit detection
* Health bar system
* Character animations
* Jump and movement mechanics
* Countdown timer
* Winner determination
* HTML5 Canvas rendering

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/bsemwal97/Fighting-game-Js.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.
